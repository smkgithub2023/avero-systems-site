# Avero Systems Landing Page

This folder contains a ready-to-deploy static landing page for `Avero Systems`.

## Files

- `index.html`
- `coming-soon.html`
- `styles.css`
- `.nojekyll`

## Easiest GitHub Pages Deployment

1. Create a separate repository for the website, such as `avero-systems-site`.
2. Copy the contents of this folder into the root of that repository.
3. Push to `main`.
4. In GitHub, open:
   `Settings -> Pages`
5. Set the source to:
   `Deploy from a branch`
6. Choose:
   `main` and `/ (root)`
7. Add your custom domain in the GitHub Pages settings once the site is live.

## Quick edits to make before launch

- Update the email address in `index.html`
- Replace the contact CTA if you want it to point to a form or scheduler instead of email
- Add a product teaser or pilot signup later when you are ready to talk about active product work
